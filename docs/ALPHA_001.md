# Ancient Utopia Quest - Alpha 0.01

## Objetivo

Crear una versión estable y jugable.

La prioridad absoluta es la estabilidad.

---

## Características Incluidas

- Joystick virtual móvil
- Movimiento suave
- Aceleración progresiva
- Desaceleración progresiva

### Cámara

- Control táctil
- Rotación horizontal
- Rotación vertical limitada
- No atraviesa objetos

### Terreno

- Procedural
- Tamaño visible limitado
- Chunks dinámicos
